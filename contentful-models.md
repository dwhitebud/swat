# Contentful Content Models

## Service
- Title (Short Text)
- Description (Long Text)
- Icon (Media)
- Slug (Short Text)
- Features (Short Text, List)
- Order (Number)

## Testimonial
- Client Name (Short Text)
- Company (Short Text)
- Quote (Long Text)
- Image (Media)
- Rating (Number)

## Team Member
- Name (Short Text)
- Position (Short Text)
- Bio (Long Text)
- Photo (Media)
- LinkedIn URL (Short Text)

## Page
- Title (Short Text)
- Slug (Short Text)
- Content (Rich Text)
- SEO Title (Short Text)
- SEO Description (Long Text)
- Featured Image (Media)
